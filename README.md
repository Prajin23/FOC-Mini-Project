## Project Overview
This mini project focuses on implementing fundamental web programming concepts using HTML, CSS, and JavaScript. It is structured as a series of practice sets designed to build a strong foundation in front-end development, covering layout design, styling techniques, responsiveness, and interactivity.

---

## Practice Sets Summary

### Practice Set 1 – Basics of CSS
- Created a div using an ID and applied background styling  
- Styled multiple headings using a common class  
- Implemented CSS using:
  - External stylesheet  
  - Internal style tag  
  - Inline styling  

### Practice Set 2 – Text Styling and Nesting
- Centered and capitalized heading text  
- Applied a global font family (Times New Roman)  
- Created nested div elements (outer and inner)  
- Adjusted font sizes for parent and child elements  

### Practice Set 3 – Shapes and Navigation
- Created a circular div using border-radius  
- Designed a basic navigation bar  

### Practice Set 4 – Page Layout
- Designed a layout with header, content area, and footer  
- Added borders to elements  
- Applied background colors with opacity  
- Defined fixed height and width for content blocks  

### Practice Set 5 – Positioning
- Positioned elements using CSS positioning  
- Applied fixed positioning to keep elements static during scroll  
- Used z-index for layering  
- Added a background image  

### Practice Set 6 – Flexbox
- Created a navigation bar using flexbox  
- Distributed elements evenly in a row  
- Centered elements using flexbox  
- Observed priority: align-self overrides align-items  

### Practice Set 7 – Media Queries
- Implemented responsive design using media queries  
- Adjusted styles based on viewport width  

### Practice Set 8 – CSS Loader
- Designed a circular loader using borders  
- Implemented rotation animation using keyframes  
- Applied continuous animation  

### Practice Set 9 – JavaScript Integration
- Created a simple webpage with HTML elements  
- Linked an external JavaScript file  

### Practice Set 10 – Interactive Webpage
- Changed text content on mouse hover  
- Restored original content on mouse out  
- Used setTimeout to display a delayed message  

### Practice Set 11 – Advanced Interactivity
- Implemented dark mode toggle  
- Added scroll-to-top button  
- Controlled button visibility based on scroll position
